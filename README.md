# Wan2.2 - Web Interface

> This is a personal implementation fork of [Wan2.2](https://github.com/Wan-Video/Wan2.2) that adds a simple web interface to run the Wan2.2 model. **For interest and educational purposes only.**

## 🙏 Acknowledgment

This project is built upon the excellent [**Wan2.2**](https://github.com/Wan-Video/Wan2.2) video generation model. All core model architecture, weights, and capabilities are from the original Wan2.2 project by the Wan team. This fork simply adds a web interface layer to make the model more accessible through a browser-based UI.

For the original Wan2.2 project, official documentation, and community resources, please visit: **[Wan2.2 GitHub Repository](https://github.com/Wan-Video/Wan2.2)**
